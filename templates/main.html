Welcome to the storage system. This is a beta unfinished version.

Shortcuts:

  • Open a new transaction - deposit
  • Open a new transaction - withdraw
  • Storage report - shows the current amount of storage
  • Traders list - showing the suppliers and costumers
  • Help page - for any help needed

Using the Storage System for the first time?

It is advised to do the following -

  1. Add a warehouse
  2. Add a supplier - this can be done from the traders page, by choosing "yes" under the is supplier property.
  3. Add items, so you can deposit and withdraw them.
  4. Change the storage, by opening new transactions. This should be done only after doing the above steps.
  5. Need more help? Check the help page .
